لتشغيل البرنامج عن بعد مع مشاركة البيانات بين جميع الأجهزة،
يجب ربطه بـ Firebase. اتبع الخطوات التالية مرة واحدة فقط:
1. اذهب إلى console.firebase.google.com → أنشئ مشروعاً جديداً
2. من المشروع → Realtime Database → Create Database → Start in test mode
3. Project Settings ← ⚙️ → Add App → Web → انسخ apiKey و databaseURL
4. الصق البيانات أدناه واضغط Connect
❌ خطأ في الاتصال. تحقق من البيانات وحاول مجدداً.
